# Block F — UAT Checklist（高效版）

> 生成 2026-06-02 by Opus 4.8。覆盖 CSV 导出/导入 + 数据管理页 + 组合删改 + 默认自愈 + 离线缓存。
> 设计原则：**一次连贯走查，少切 App 状态**。发现问题带「截图 + 所在屏 + 复现步骤」回来集中修。

## ⚠️ 开始前必读

- **环境**：用 **Clean Env**（`+arc-clean`）跑大部分；组合删改/导入这种写操作别在 Real Env 乱试（除非你想清数据）。
- **🔴 离线缓存（D 区 OF.1–12）必须用 dev build，Expo Go 跑不了**：MMKV 用 NitroModules，Expo Go 里 persister 直接返回 null（静默退化成无缓存）。所以 **A/B/C 区现在就能测；D 区等你 EAS dev build 上机后再测**（时序第 3 步之后）。这不是 bug，是 Expo Go 的固有限制。
- 中英各扫一遍（至少切一次 en 看文案）。
- `pnpm mobile -- --clear` 冷启动，避免旧 bundle。
